# 🍽️ Avaro Restaurant Website

A modern restaurant website I’m building for **Avaro**, focused on clean design, smooth user experience, and real-world features.


## ✨ Overview

This project is part of my frontend and backend learning journey where I’m trying to build something that feels like a real product—not just a basic project.



## 🚀 Features

- Responsive layout (mobile-friendly)
- Restaurant-style UI design
- Menu section
- Booking call-to-action
- Contact form


## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
